# imagepicker
Image Picker Library

## Maven
Step 1. Add the JitPack repository to your build file
```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

## Gradle
Step 2. Add the dependency
```
dependencies {
	        compile 'com.github.camstarts:imagepicker:v1.0.2'
	}
```
